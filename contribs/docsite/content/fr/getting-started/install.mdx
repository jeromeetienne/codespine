# Installation

`codespine` est publié sur npm sous le nom
[`codespine`](https://www.npmjs.com/package/codespine). Chaque
commande de cette documentation s'exécute via `npx`, il n'y a donc **aucune installation globale** à
gérer — vous pointez le binaire vers n'importe quel projet TypeScript et c'est parti.

## L'exécuter dans votre agent de codage

Installez les ressources [Claude Code](https://claude.com/claude-code)
empaquetées — les commandes slash `/codespine-optimize` et `/codespine-interview`
et la compétence `codespine-query` — dans le répertoire `.claude/` de votre projet
afin que l'agent puisse piloter le graphe :

```bash
npx codespine install
```

Il n'y a aucune clé d'API ni aucun fournisseur de LLM à configurer :
l'environnement d'exécution de l'agent est Claude Code lui-même. Voir
[`install`](/commands/install) pour les options et
[Commandes slash de l'agent](/agent/slash-commands) pour le rôle de chaque commande.

## Prérequis

- **Node.js ≥ 20.12** (à vérifier avec `node --version`).
- Le projet cible doit posséder un **`tsconfig.json`** — l'extracteur le charge
  via `ts-morph` à partir de cette configuration.
- **Pour l'agent d'optimisation uniquement :** [Claude Code](https://claude.com/claude-code).
  L'agent est livré sous la forme de la commande slash `/codespine-optimize`, il n'y a donc aucune clé d'API
  ni aucun fournisseur de LLM à configurer — l'environnement d'exécution de l'agent est Claude Code lui-même.

## L'exécuter sur votre propre projet

Aucune étape d'installation. Depuis la racine de n'importe quel projet doté d'un `tsconfig.json` :

```bash
npx codespine extract . --semantic
npx codespine load
npx codespine dead-exports
```

`npx` télécharge le binaire publié à la première utilisation et le met en cache. Voir
[Le pipeline](/getting-started/pipeline) pour la présentation complète.

## Cloner le dépôt

Pour essayer l'outil sur son propre code source, ou pour ajouter votre propre parcours, clonez le dépôt
et installez ses dépendances :

```bash
git clone https://github.com/jeromeetienne/codespine.git
cd codespine
npm install
```

Depuis un clone, vous pouvez exécuter la CLI au moyen des scripts npm fournis
(`npm run extract -- . --semantic`, `npm run dev -- load`, …) ou, comme le fait la documentation,
avec `npx codespine <command>`.
